MasOS用のショートカットです。

### 凡例

| 表記 | 対応するキー |
| ---- | ---- |
|⌘|Cmd|
|⌥|Option|
|⇧|Shift|
|^|Ctrl|

### 基本操作

| ショートカット | アクション |
| ---- | ---- |
| `⌘N` | 新規ファイル |
| `⌘O` | フォルダを開く |
| `⌘S` | 保存 |
| `⌘⇧S` | 名前を付けて保存 |
| `⌘Q` | Zedを終了 |
| `⌘,` | 設定を開く |
| `⌘K⌘S` | ショートカット設定を開く |
| `⌘⇧W` | 現在のウィンドウを閉じる |
| `⌘⇧N` | 新しいウィンドウを開く |

### 編集

| ショートカット | アクション |
| ---- | ---- |
| `⌘Z` | 元に戻す |
| `⌘⇧Z` | やり直し |
| `⌘X` | 切り取り |
| `⌘C` | コピー |
| `⌘V` | 貼り付け |
| `⌘A` | すべて選択 |
| `⌘/` | 行コメントの切り替え |

### 行の操作

| ショートカット | アクション |
| ---- | ---- |
| `⌥↑` | 行を上に移動 |
| `⌥↓` | 行を下に移動 |
| `⌥⇧↑` | 行を上に複製 |
| `⌥⇧↓` | 行を下に複製 |
| `⌘⇧K` | 行を削除 |

### マルチカーソル・選択

| ショートカット | アクション |
| ---- | ---- |
| `⌘⇧L` | すべての同じ選択を選択 |
| `⌘⌥↑` | 上にカーソルを追加 |
| `⌘⌥↓` | 下にカーソルを追加 |
| `^⇧→` | 選択範囲を拡張 |
| `^⇧←` | 選択範囲を縮小 |

### 検索・置換

| ショートカット | アクション |
| ---- | ---- |
| `⌘F` | バッファ内検索 |
| `⌘⇧F` | プロジェクト内検索 |
| `⌘⌥F` | 置換 |
| `⌘G` | 次のマッチを選択 |
| `⌘⇧G` | 前のマッチを選択 |

### タブ操作

| ショートカット | アクション |
| ---- | ---- |
| `⌘{` | 前のタブに移動 |
| `⌘}` | 次のタブに移動 |
| `⌥⌘←` | 前のタブに移動 |
| `⌥⌘→` | 次のタブに移動 |
| `^1〜^9` | 1〜9番目のタブに移動 |
| `^0` | 最後のタブに移動 |

### 分割・ペイン

| ショートカット | アクション |
| ---- | ---- |
| `⌘\` | 右に分割 |
| `⌘K ↑` | 上に分割 |
| `⌘K ↓` | 下に分割 |
| `⌘K ←` | 左に分割 |
| `⌘K →` | 右に分割 |

### パネル・ドック

| ショートカット | アクション |
| ---- | ---- |
| `⌘B` | 左ドックの切り替え |
| `⌘J` | 下ドックの切り替え |
| `⌘⇧B` | アウトラインパネル |
| `^~` | ターミナル |

### コード編集・ナビゲーション

| ショートカット | アクション |
| ---- | ---- |
| `F12` | 定義に移動 |
| `⌥F12` | 定義を分割で開く |
| `⇧F12` | 実装に移動 |
| `F2` | 名前変更 |
| `⌘.` | コードアクション |
| `^Space` | 補完表示 |
| `⌘⇧I` | コードをフォーマット |

### AI・エージェント機能

| ショートカット | アクション |
| ---- | ---- |
| `^↵` | インラインアシスト |
| `⌘>` | 選択範囲を引用 |

## ピアノロール

### 基本編集

| 操作            | ショートカット              |
| ------------- | -------------------- |
| ピアノロールを開く | `F7`                 |
| ノート描画         | `P`                  |
| ペイント          | `B`                  |
| カット           | `C`                  |
| 消しゴム          | `Delete` または `右クリック` |
| ノート選択         | `⌘ + Drag`           |
| 全選択           | `⌘ + A`              |
| 選択解除          | `⌘ + D`              |
| コピー           | `⌘ + C`              |
| 貼り付け          | `⌘ + V`              |
| 複製            | `⌘ + B`              |
| 元に戻す          | `⌘ + Z`              |
| やり直し          | `⌘ + ⇧ + Z`          |

### ノート移動・長さ調整

| 操作        | ショートカット         |
| --------- | --------------- |
| ノート移動     | `Drag`          |
| 微調整移動     | `⌘ + Drag`      |
| ノート長変更    | ノート端をDrag       |
| ベロシティ変更   | 下部VelocityをDrag |
| 複数ノート同時移動 | `⇧ + Drag`      |
| ノート削除     | `Delete`        |

### 高速入力系

| 操作       | ショートカット         |
| -------- | --------------- |
| クオンタイズ   | `⌘ + Q`         |
| クイックレガート | `⌘ + L`         |
| ストラム     | `⌥ + S`         |
| アルペジエーター | `⌥ + A`         |
| Humanize | `⌥ + R`         |
| Flam     | `⌥ + F`         |
| Chopper  | `⌥ + U`         |
| ランダマイズ   | `⌥ + R`         |
| スケール強制   | `Stamp Tool` 使用 |

### ズーム・ナビゲーション

| 操作      | ショートカット       |
| ------- | ------------- |
| 横ズーム    | `⌘ + マウスホイール` |
| 縦ズーム    | `⌥ + マウスホイール` |
| 全体表示    | `Backspace`   |
| スクロール   | `⇧ + マウスホイール` |
| 再生位置へ移動 | `0`           |


### MIDI編集高度操作

| 操作        | ショートカット           |
| --------- | ----------------- |
| ゴーストノート表示 | `⌥ + V`           |
| ノートスライド   | Slide Note使用      |
| ポルタメント    | Portamento Note使用 |
| ミュートノート   | `Alt` モード使用       |
| ノートカラー変更  | `1〜9`             |
| ノート分割     | `⌥ + U`           |
| ノート結合     | `⌘ + G`           |

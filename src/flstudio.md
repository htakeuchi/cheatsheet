## ピアノロール

### 基本編集

| 操作            | ショートカット              |
| ------------- | -------------------- |
| ピアノロールを開く | `F7`                 |
| 描画ツール         | `P`                  |
| ペイント          | `B`                  |
| スライスツール       | `C`                  |
| 削除ツール         | `D`                  |
| ノート削除         | `fn + Delete` または `右クリック` |
| ノート選択         | `⌘ + Click`          |
| 選択に追加         | `⌘ + ⇧ + Click`      |
| 全選択           | `⌘ + A`              |
| 選択解除          | `⌘ + D`              |
| コピー           | `⌘ + C`              |
| 貼り付け          | `⌘ + V`              |
| 複製            | `⌘ + B`              |
| 元に戻す / やり直し | `⌘ + Z`              |
| 段階的に元に戻す     | `⌘ + ⌥ + Z`          |
| カット           | `⌘ + X`              |

### ノート移動・長さ調整

| 操作        | ショートカット         |
| --------- | --------------- |
| ノート移動     | `Drag`          |
| スナップ無視移動 | `⌥ + Drag`      |
| ナッジ移動     | `⌥ + 矢印キー`      |
| ノート長変更    | ノート端をDrag       |
| ノートプロパティ変更 | `⌥ + マウスホイール`  |
| ベロシティ変更   | 下部VelocityをDrag |
| 複数ノート同時移動 | 選択してDrag        |
| ノート削除     | `fn + Delete` または `右クリック` |

| 画面パン       | `Middle Click + Drag`  |
| グローバルスナップ切替 | `Delete (Backspace)`    |

## 覚えておくと便利な操作

| 操作                | 内容          |
| ----------------- | ----------- |
| `⇧ + H`           | ノートデータのある次チャンネルへ |
| `Middle Click + Drag` | 画面パン       |
| `⌥ + Drag`        | スナップ無視      |
| `Right Click`     | 消去          |
| `⌥ + Mouse Wheel` | 近くのノートプロパティ変更 |
| `Ctrl + ⌥ + Mouse Wheel` | ノートプロパティを微調整 |
